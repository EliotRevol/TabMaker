# Tab Spectro Helper — Pack (modulaire)

## Lancer
```bash
pip install -r requirements.txt
python run.py
```

## MP3
Les MP3 nécessitent **ffmpeg** (installé et accessible dans le PATH) + `pydub`.

## Contrôles
- File > Open : charger un audio
- Clic droit sur spectrogramme : ajouter une croix (note estimée dans Notes)
- View > Guitar View : ouvre la vue manche (grille), bouton ▶ Play joue les notes sélectionnées
- Tools > Toggle Mic : lignes vertes selon fréquences détectées
- Ctrl + scroll : zoom ; Shift + 2 doigts : scroll horizontal si ton touchpad n'envoie pas X
